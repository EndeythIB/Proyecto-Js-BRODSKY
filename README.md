<section class="trabajo">
	<h3>Visión general del proyecto</h3>
	<p>Este proyecto simula ser una librería virtual que es capaz de distinguir tres tipos de usuarios que cuentan con distintos privilegios:</p>
	<ol>
		<li>Admin: Se accede a este mediante el incio de sesión (usuario: "admin" contraseña "123") puede agregar nuevos productos al sitio. Para ello cuenta con el botón "Agregar item" en la barra de navegación.</li>
		<li>Cliente: Se accede registrandose e iniciando sesión con los datos elegidos, puede agregar artículos al carrito y luego visualizarlos con el botón que aparece en la barra de navegación.</li>
		<li>Invitado: Usuario nuevo a la página, solamente puede ver los productos.</li>
</ol>

</section>

<section id="instrucciones" class="instrucciones">
<h2>Instrucciones de uso</h2>
<p>A continuación se detallará de qué manera poder acceder al sitio en cada uno de los modos de usuario:</p>
<ol><li>Modo admin: El usuario deberá loguearse con el nombre de usuario "admin" y la contraseña "123"</li>
	<h3>Como agregar un nuevo producto</h3>
	<p>Para agregar un nuevo producto (función especial del modo admin) simplemente se debe hacer click en el boton "Agregar item" lo cual desplegará tres imputs, nombre, precio, url de imagen, una vez completados estos campos con la información deseada (se podra ver una preview de la Card mientras se completan los datos, para ver si es o no del agrado del admin antes de agregarla) apretar el botón "Agregar" y listo.</p>
	<li>Modo invitado: Cualquier usuario que entre por primera vez a la página o no se haya logueado sera entrará en modo invitado</li>
	<li>Modo cliente: El usuario deberá registra sus datos y luego loguearse para poder acceder a las compras del sitio.</li>
</section>
